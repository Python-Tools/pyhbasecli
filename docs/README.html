pyhbasecli¶

基于thrift2的hbase客户端

  • author: hsz

keywords: hbase,thrift

特性¶

  • 可以连接处理阿里云hbase

  • 使用的是thrift2版本

  • 支持http方式连接

  • python友好的接口设计

安装¶

pip install pyhbasecli

例子¶

with HBaseCli(
                url=URL,
                headers={"ACCESSKEYID": "root", "ACCESSSIGNATURE": "root"}) as client:
    nsinfo = client.show_namespaces()
    print(nsinfo)
